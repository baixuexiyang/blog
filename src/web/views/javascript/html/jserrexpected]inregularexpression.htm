正则表达式中缺少 ']'

试图创建一个用于正则表达式匹配的字符类，但是没有右中括号。单个的文字字符组合可以用括号括起来表示一个字符类。一个字符类可以匹配其中包含的任一字符。例如，/[abc]/ 可以匹配字母 “a”, “b”, 或 “c” 的任何一个。

要改正此错误

  • 给正则表达式加上右中括号。

注意   如果想匹配单中括号，可以使用反斜杠将其转义 - \( - 这样就不会被JScript 解释为特殊字符。)
