Add 方法 (Folders)

向 Folders 集合中添加新的 Folder 。

object.Add (folderName)

参数

object

必选项。 应为 Folders 集合的名称。

folderName

必选项。 要添加的新 Folder 的名称。

说明

下面的例子说明了使用 Add 方法创建新文件夹的方法。 

function AddNewFolder(path,folderName)
{
   var fso, f, fc, nf;
   fso = new ActiveXObject("Scripting.FileSystemObject");
   f = fso.GetFolder(path);
   fc = f.SubFolders;
   if (folderName != "" )
      nf = fc.Add(folderName);
   else
      nf = fc.Add("New Folder");
}

如果 folderName 已经存在了，则会出错。

请参阅

Add 方法 (Dictionary)应用于： Folders 集合
