length 属性 (String)

返回 String 对象的长度。

strVariable.length
"String Literal".length

说明

length 属性包含一个整数，用来指出 String 对象中的字符数。String 对象中的最后一个字符的索引为 length - 1。

要求

版本 1

请参阅

应用于： String 对象
