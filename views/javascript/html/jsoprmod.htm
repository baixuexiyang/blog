取余运算符 (%)

一个表达式的值除以另一个表达式的值，返回余数。

result = number1 % number2

参数

result

任何变量。

number1

任何数值表达式。

number2

任何数值表达式。

说明

取余（或余数）运算符用 number1 除以 number2 （把浮点数四舍五入为整数），然后只返回余数作为 result。例如，在下面的表达式中，A （即 result）等于 5。 

A = 19 % 6.7

要求

版本 1
