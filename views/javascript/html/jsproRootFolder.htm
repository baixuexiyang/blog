RootFolder 属性

返回一个 Folder 对象，表示指定驱动器的根文件夹。 只读。

object.RootFolder

object 总是为 Drive 对象。

说明

可以通过返回的 Folder 对象来访问驱动器上的所有文件和文件夹。

下面的例子说明了 RootFolder 属性的用法： 

function GetRootFolder(drv)
{
   var fso,d;
   fso = new ActiveXObject("Scripting.FileSystemObject");
   if (fso.DriveExists(drv))
      {
         d = fso.GetDrive(drv);
         return(d.RootFolder);
      }
   else
      return(false);
}
